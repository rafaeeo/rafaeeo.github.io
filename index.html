Principais Projetos

Projeto 1: Análise de Vendas

Dashboard interativo em Power BI para acompanhar indicadores de vendas, incluindo faturamento, ticket médio, taxa de crescimento e visualização dos dados.
Ferramentas: Power BI, Excel.

Projeto 2: Análise Agro

Análise detalhada e dashboard interativo dos dados de produção de sementes que inclui a exploração de variabilidade, a distribuição dos dados e sugestões para planejamento e intervenção.
Ferramentas: Python, scikit-learn, Pandas, SQL.

Projeto 3: Predição da Resistência à Compressão do Concreto

Neste projeto analisei o dataset "Concrete Compressive Strength" do UCI Machine Learning Repository para entender os fatores que influenciam a resistência à compressão do concreto e construir um modelo de regressão linear múltipla para prever essa resistência.
Ferramentas: Python, scikit-learn, Pandas, MatPlotLib.

Projeto 4: Análise de Fatores que Influenciam os Preços de Imóveis

Este projeto utiliza o dataset "House Prices - São Paulo" do Kaggle para identificar e analisar os fatores que influenciam os preços dos imóveis na cidade de São Paulo. O objetivo é aplicar técnicas de pré-processamento, análise exploratória, modelagem preditiva e avaliação para construir um modelo que explique e preveja os preços dos imóveis.
Ferramentas: Python, scikit-learn, Pandas, Seaborn.
